# Rhino Runner

Яркий браузерный раннер в духе Chrome Dino, но в более безумной пустынной версии: вместо динозавра здесь бежит антропоморфный носорог в желтой куртке и красных штанах.

**Играть онлайн:** [abroskinp.github.io/rhino-runner](https://abroskinp.github.io/rhino-runner/)

![Rhino Runner gameplay screenshot](./assets/preview.svg)

## Об игре

Rhino Runner - это быстрая аркада про бег, прыжки и сбор лайков в жаркой пустыне.  
Носорог автоматически несется вперед, перепрыгивает кактусы, живые какашки и пустынных грифов, собирает сердечки в воздухе и включает `LIKE POWER` за каждые 10 лайков.

## Что есть в игре

- бесконечный раннер с ростом скорости
- пустынный сеттинг с дневным и ночным режимом
- несколько видов кактусов, живые какашки и грифы вместо обычных птиц
- лайки-сердечки, которые нужно собирать в прыжке
- отдельный счетчик лайков в интерфейсе
- бонус `LIKE POWER` за каждые 10 лайков: 3 секунды неуязвимости и ускорение на 33%
- таймер активного бонуса на экране
- мобильное управление через тап по экрану
- звук прыжка, шагов, столкновений, награды и праздника за каждые 10 лайков
- публикация через GitHub Pages с автоматическим обновлением после `git push`

## Управление

- На десктопе:
  - `Space` или `ArrowUp` - прыжок
  - после проигрыша `Space` - начать заново
- На телефоне:
  - тап по экрану - прыжок и старт
  - после проигрыша тап по экрану - начать заново

## Что изменилось в текущей версии

- переработан стиль пустыни: горы, облака, день и ночь стали глубже и атмосфернее
- носорог получил более выразительный силуэт и живую анимацию
- сердечки стали крупнее и чище по форме
- препятствия теперь лучше вписаны в сеттинг: кактусы разнообразнее, а птицы заменены на грифов
- камни заменены на смешные живые какашки с разными формами

## Публикация

Проект настроен так, что GitHub Pages обновляется автоматически после отправки изменений в ветку `main`.

Обычный порядок такой:

```bash
git add .
git commit -m "Update game"
git push
```

После этого GitHub сам пересоберет игру и обновит сайт по адресу:

[https://abroskinp.github.io/rhino-runner/](https://abroskinp.github.io/rhino-runner/)
